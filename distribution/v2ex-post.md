# V2EX 帖子：创造节点

## 标题
你的 AI 写东西一股机器人味？这个开源 Skill 一刀切掉

## 正文

先看个东西。下面这段话，你闻出 AI 味了没：

> 在当今快速变化的时代，企业需要拥抱不确定性，通过系统性创新在挑战中寻找机遇。这不仅仅是技术问题，更是一个深层次的战略转型。

闻出来了吗。"在当今...的时代"+"不仅仅是...更是..."+"系统性..."——三连 AI 暴击。

再看去完 AI 味的版本：

> 时代变得快。企业跟不上就被吃掉。这不是技术问题，是生存问题。

**三句话。有骨头。像人说的。**

---

### 这是怎么做到的

GitHub 上有个项目叫 [stop-slop](https://github.com/hardikpandya/stop-slop)，9200+ Star。一个纯 Markdown 文件——教 AI 识别"自己的写作口癖"然后删掉。

但它只支持英文。中文 AI 写作的问题比英文严重十倍：
- "在当今...的时代" ——万能开头，AI 用了无数次
- "不仅仅是...更是..." ——二元对立的 Chinese ver.
- 四字成语叠叠乐 —— 一段塞三个，读者直接划走
- "毋庸置疑 / 众所周知 / 综上所述" —— 全是废话
- 每段结尾必须押一个金句 —— 假得要命

我 fork 了原版，加了 **12 条中文专属反模式**，做成第一个双语去 AI 味 Claude Skill。

---

### 怎么用

```bash
git clone https://github.com/tomerose/stop-slop.git \
  ~/.claude/skills/stop-slop
```

重启 Claude Code。之后你写的每段文字，AI 自动用 **5 维打分**检查：
- 直接性：在陈述还是在宣告？
- 节奏：句子长短有变化吗？
- 信任：尊重读者智商吗？
- 真实感：听起来像人吗？
- 密度：还有可删的吗？

**低于 35/50 分：打回重写。目标 45+ 分交付。**

支持 Claude Code / Cursor / Copilot。

---

### 为什么值得你花 2 分钟装

2026 年有两个事实：

1. **82% 的创作者每天用 AI 写东西。**
2. **76% 的读者能在 3 秒内嗅出 AI 味。**

你的公众号文章、产品文案、小红书笔记、推文——读者扫一眼就知道是不是 AI 写的。不是他们讨厌 AI，是 AI 味让人不信任。"看起来像人写的"正在变成一种稀缺能力。

stop-slop 做的事很简单：**在 AI 输出到达读者之前，把 AI 味砍掉。**
